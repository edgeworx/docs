# Get Started: Edgeworx Cloud

## Build, deploy and run AI from cloud to Edge.

Getting started with AI at the edge has never been easier. Using Edgeworx Cloud and your favorite Coral-based Edge node \(more node types supported soon!\) you can get your node deployed at the Edge in just a few minutes. Easily protoype in the lab, then deploy it to the real world without having to change any code.
