# Applications

Applications are a group of microservices bundled to work together. Apps are defined using YAML files and can be deployed and updated by uploading those YAML files

They can be managed through the [portal](../get-started-ec/get-started-deploy-app.md#deploy-a-custom-app-using-the-portal)  or using [edgectl](../get-started-edgectl/#deploy-an-application)
