# Underlying Open-Source Technologies

* [**Eclipse ioFog**](https://iofog.org)**:** Darcy uses Eclipse ioFog to deploy and manage nodes and apps at the Edge
* [**DevicePlane**](https://deviceplane.com)**:** SSH into nodes at the Edge from anywhere
* [**Docker**](https://docker.com)**:** Container runtime and tools for building applications
* [**Docker Hub**](https://hub.docker.com)**:** A great place for uploading, storing, and hosting your Darcy AI application containers
