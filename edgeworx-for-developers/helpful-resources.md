# Other Helpful Resources

## Underlying Open-source Technologies

* [**Eclipse ioFog**](https://iofog.org)**:** Darcy uses Eclipse ioFog to deploy and manage nodes and apps at the Edge
* [**DevicePlane**](https://deviceplane.com)**:** SSH into nodes at the Edge from anywhere
* [**Docker**](https://docker.com)**:** Container runtime and tools for building applications
* [**Docker Hub**](https://hub.docker.com)**:** A great place for uploading, storing, and hosting your Darcy AI application containers

## AI Development

* [**OpenCV**](https://opencv.org)**:** _OpenCV_ provides a real-time optimized Computer Vision library, tools, and hardware. It also supports model execution for Machine Learning \(ML\) and Artificial Intelligence \(AI\)
* [**TensorFlow:** ](https://www.tensorflow.org/community)Whether you’re an expert or a beginner, TensorFlow is an end-to-end platform that makes it easy for you to build and deploy ML models.
